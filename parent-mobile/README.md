# Sukat Kalusugan Parent Mobile

Expo demo app for the parent-facing mobile experience.

## Screens

- Login screen
- Dashboard
- Child list
- Child profile
- Growth history
- Notifications
- Nutritional status results

## Run it

1. Install dependencies inside this folder.
2. Start the Expo server with `npm run start`.
3. Open it on a phone with Expo Go or run an emulator.

## Notes

- This is a mobile demo shell with mock data.
- It mirrors the existing nutrition UI language while keeping the navigation phone-friendly.
